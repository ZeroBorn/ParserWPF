Программа предназначена для конвертации xml файла в графическое изображение.

Для использования необходимо:

1) нажать кнопку "Добавить значения", что бы загрузить в бд сокращенные и полные наименования тегов;
2) нажать на "Обзор документа", и выбрать файл типа xml;
3) далее нажать "Преобразовать документ", что отобразит файл в графическом изображении со всеми полными названиями, и аттрибутами.
